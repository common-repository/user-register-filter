= Features included: =
* Deny registration if email ends with...
* Smart filtering (detects suspicious behaviours)

== Installation ==

Install like any other plugin. Don't forget to activate it!

1. Upload the plugin files to the `/wp-content/plugins/tt-user-register-filter` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Use the *User Register Filter* screen to configure the plugin.
4. That's all. No necessary changes to .htaccess or other config files.
